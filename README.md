# AJAX CMS Admin Panel

A simple AJAX-based Admin Dashboard built using PHP, MySQL, and JavaScript.  
This project demonstrates how backend operations can be performed without page reload, improving user experience similar to modern CMS platforms.

---

## 🚀 Features

- User Management
  - Add user (AJAX)
  - Edit user (AJAX)
  - Delete user (AJAX)

- Post Management
  - Add post (AJAX)
  - Edit post (AJAX)
  - Delete post (AJAX)

- Dashboard
  - Overview of users and posts

- No Page Reload
  - All operations use AJAX for smooth interaction

---

## 🛠️ Tech Stack

- PHP
- MySQL
- JavaScript (AJAX / jQuery)
- HTML & CSS

---

## 🎯 Purpose

This project was built to demonstrate asynchronous CRUD operations and backend interactivity improvements, similar to admin panels in CMS platforms like Joomla.

## ⚙️ Setup Instructions

1. Clone the repository
2. Import the database (SQL file)
3. Configure database connection in `db.php`
4. Run the project on local server (XAMPP/Laragon)
